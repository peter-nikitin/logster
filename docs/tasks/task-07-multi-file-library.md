# Task 07: Multi-File Library

## Product Increment

When I have several files, I can browse them as a local library and switch between them easily.

## Demo Result

Open the app and verify:

- multiple stored datasets are listed
- switching between datasets is fast
- deleting one dataset does not affect the others

## Goal

Prepare the product for comparison workflows by making multiple datasets manageable.

## Scope

- improve dataset library UI
- sort and label stored datasets
- show file metadata such as import time and row count

## Acceptance Criteria

- the user can understand what datasets are stored locally
- switching datasets is clear and predictable
- dataset metadata helps identify the right file
