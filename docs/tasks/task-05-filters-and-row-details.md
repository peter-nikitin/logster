# Task 05: Filters And Row Details

## Product Increment

When I have a loaded dataset, I can narrow it down by method or text and inspect a selected row in detail.

## Demo Result

Open the app and verify:

- method filter works
- text search works
- selecting a row shows detailed payload data

## Goal

Make the table practical for investigation, not only for browsing.

## Scope

- method filter
- free-text search on message
- payload-only filter
- improved row details panel

## Acceptance Criteria

- filtering by method like `[use-tab-content]` works
- text search updates the visible rows
- payload-only filter updates the visible rows
- row details show formatted payload and raw row
