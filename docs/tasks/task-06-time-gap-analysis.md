# Task 06: Time Gap Analysis

## Product Increment

When I inspect a dataset, I can quickly see where the biggest time gaps are.

## Demo Result

Open the app and verify:

- large deltas are visually highlighted
- summary metrics show the biggest delay and basic dataset stats

## Goal

Turn the viewer into a timing-analysis tool for one file.

## Scope

- delta highlighting
- summary metrics
- biggest-gap surfacing

## Acceptance Criteria

- slow gaps are easy to spot in the table
- summary metrics update for the active dataset
- the biggest delta is visible without scanning every row manually
