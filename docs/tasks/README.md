# Main Path Tasks

These tasks are organized as product increments.

Rule:

- after each task, the product owner should be able to open the app and see a concrete new result

Main path:

1. Open the app.
2. See bundled log files.
3. Open one file and inspect parsed data.
4. See the same data in a readable table.
5. Upload a custom file.
6. Keep uploaded files in browser storage.
7. Reopen stored files later without uploading again.

Current increment order:

1. `task-01-bundled-files-and-parsed-preview.md`
2. `task-02-table-view-for-active-dataset.md`
3. `task-03-custom-file-upload.md`
4. `task-04-browser-storage-and-restore.md`
5. `task-05-filters-and-row-details.md`
6. `task-06-time-gap-analysis.md`
7. `task-07-multi-file-library.md`
8. `task-08-resilience-and-empty-states.md`

The first four tasks match the visible milestones defined by the product owner.
